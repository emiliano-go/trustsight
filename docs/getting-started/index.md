# Getting Started

<img src="../assets/images/trustsight-banner.png" alt="TrustSight" width="700"/>

The fastest path from zero to your first `trustsight review`. Follow these pages in order.

<div class="grid cards" markdown>

-   [**Installation**](installation.md)

    Install TrustSight via pip, AUR, or from source. Configure an LLM provider if you want English verdict translations.

-   [**Quickstart**](quickstart.md)

    Run your first review, read the output table, and understand what each column means.

-   [**Reading a Report**](reading-a-report.md)

    Deep dive into every section of the inspect output: score breakdown, evidence tiers, rule firings, and novelty context.

</div>

Once you have completed the getting-started path, explore the [Guides](../guides/index.md) for specific workflows (CI integration, flag investigation, configuration tuning) and the [Explanation](../explanation/index.md) for the reasoning behind the scoring model.
